# Sortable Items List

## v1.2

### Requirements:

1. Docker
2. Docker Compose

##

### Installation instructions:

1. Open a terminal.
2. Run the commands below:

```
git clone https://github.com/ezequiel-saila/sort-items-list.git
```
```
cd sort-items-list
```
```
docker-compose build 
```
```
docker-compose up -d
```

## Site url
https://localhost:8000
